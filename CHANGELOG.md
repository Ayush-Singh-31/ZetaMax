# Changelog

All notable changes to ZetaMax are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and released versions will follow [Semantic Versioning](https://semver.org/). **V01** is the project’s first public-release label; the Xcode target currently uses marketing version `1.0` and build `1`.

## [V01] - Unreleased

### Added

- Native macOS practice flow with Classic, Adaptive, Targeted, and Benchmark modes.
- Configurable addition, subtraction, multiplication, and exact-division drills.
- Targeted practice for two-digit multiplication, exact division, negative subtraction, squares and cubes, percentages, decimal arithmetic, and quant interview preparation.
- Seeded, deterministic question generation.
- Versioned benchmark profiles at 30 seconds, 1, 2, 5, and 10 minutes.
- On-device SwiftData persistence for sessions, attempts, submissions, and rebuildable skill estimates.
- Adaptive sampling based on relative timing, recent slowdown, recency, and uncertainty.
- Personal recommendations with one-click focused session preparation.
- Analytics for trends, operations, categories, response-time distribution, session pace, operand patterns, slow completions, and benchmark readiness.
- Searchable, responsive history with compact list/detail navigation.
- CSV and JSON export with a versioned schema.
- System, light, and dark appearance options.
- Recovery for sessions interrupted by sleep or application termination.
- Unit, integration, analytics-service, performance-fixture, accessibility, responsive-layout, and UI coverage.

### Security and privacy

- App Sandbox and hardened runtime enabled.
- User-selected file access limited to export destinations.
- No accounts, application backend, third-party SDKs, ads, or tracking.
- Complete local-data deletion available in Settings.

- Add an easily accessible privacy-policy link inside the application.
- Complete distribution signing, App Store screenshots, metadata, archive validation, and review submission.
